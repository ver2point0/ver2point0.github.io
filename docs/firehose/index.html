Temperature Conversion

This ruby program will prompt the user for a temperature in degrees Celsius and let the user know what the corresponding temperature is in Fahrenheit. 


puts "########################################"
puts "Conver Celsius to Fahrenheit (°C to °F)"
puts "########################################"

puts "Enter degrees(°) in Celsius: "
celsius_degrees = gets.chomp.to_f

def celsius_to_fahrenheit(c_degrees)
  return (c_degrees * 9/5) + 32
end

puts "#{celsius_degrees} °C is #{celsius_to_fahrenheit(celsius_degrees)} °F."

Foobar

This ruby program is an adaptation of a classic technical interview problem which displays sequences of the Foobar pattern. 


puts "########################################"
puts "Foobar challenge"
puts "########################################"

puts "How many items do you want to see?"

items = gets.to_i

def foobar(n)
  
  divisible_by_3 = n % 3
  divisible_by_5 = n % 5
  
  if (divisible_by_3 == 0 && divisible_by_5 == 0)
    "Foobar"
  elsif divisible_by_5 == 0
    "Bar"
  elsif divisible_by_3 == 0
    "Foo"
  else
    "#{n}"
  end

end

(1..items).each do |i|
  print "#{foobar(i)} "
end
